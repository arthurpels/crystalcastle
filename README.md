# crystalcastle

**Движок:** Unity 2022.3 LTS (3D URP)

## Состав команды
* Миша
* Артур
* Курбан
* Владимир

## Инструкция по запуску
1. Клонируйте репозиторий: `git clone <ссылка_на_ваш_репозиторий>`
2. Откройте Unity Hub и нажмите **Add project**.
3. Укажите путь к склонированной папке `CrystalCastle`.
4. Убедитесь, что Editor Version установлена на **2022.3 LTS**.
5. Откройте проект и дождитесь завершения импорта ассетов.
6. Стартовая сцена находится в `Assets/Scenes/`.
